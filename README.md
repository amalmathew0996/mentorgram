# Mentorgram AI

AI-Powered Global Mentoring Platform for Education and Careers.

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Create a `.env.local` file and add your Anthropic API key:
   ```
   VITE_ANTHROPIC_API_KEY=your_key_here
   ```
   Get your key from: https://console.anthropic.com

3. Run locally:
   ```
   npm run dev
   ```

4. Build for production:
   ```
   npm run build
   ```

## Deploy to Vercel

1. Push this project to GitHub
2. Go to vercel.com and import your GitHub repo
3. Add environment variable: `VITE_ANTHROPIC_API_KEY`
4. Deploy!
5. Connect your custom domain: mentorgramai.com
